Overview:

Link to the wiki article

This is by no means an original program. A friend of mine brought up the Game of Life in conversation and I started thinking about my own implementation based on Conway's original rules. I realised that I had actually encountered the GoL before, in my second year of university. However much of the code for that assignment was already provided, and it was all console based with a static starting colony. This version is graphical and multi-threaded but stays true to the original sim.

Feature List:

  • Exact replication of the original Game of Life devised by Conway in 1970
  • Visual representation of colony on a 2D-grid
  • Create/Destroy lifeforms at any time, even while the simulation is running
  • Several loadable templates
